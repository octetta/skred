## Ubuntu 24.04

```
sudo apt install build-essential
sudo apt install libasound2-dev
sudo apt install tcl tk tcl-udp
sudo apt install wine64 mingw-w64
```

## Fedora 42

```
sudo dnf install gcc g++ make autoconf binutils gdb
sudo dnf install alsa-lib alsa-lib-devel
sudo dnf install tcl8 tcl8-devel tcl-tcludp tk8 tk8-devel
sudo dnf install wine
sudo dnf install mingw64-gcc-c++ mingw32-gcc-c++
```
